# Docs

## Usage

Make sure you have Semantic UI css in your project.

Run `npm install @hivebeat/kenai`.

You can import components like this:
```javascript
import { Card, Form, Input } from '@hivebeat/kenai'

const MyComponent = () => (
  <Card>
    <Form>
      <Input type='text' value='my name' title='name' />
    </Form>
  </Card>
)
```
